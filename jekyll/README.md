
* Install rbenv
  + Ubuntu: apt-get install rbenv
  + MacOS: brew install rbenv ruby-build
* Install ruby-build
  + git clone https://github.com/sstephenson/ruby-build.git ~/.rbenv/plugins/ruby-build
* Install the right ruby
  + rbenv install 2.2.0
* Install bundler
  + gem install bundler
  + rbenv rehash
* Install the gems
  + bundle install
  + rbenv rehash
* check jekyll
  + cd ..
  + jekyll/run-jekyll.sh
